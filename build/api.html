API

Import the package:

import pySTIM as pst

Visualization

plot_scatter(adata[, xlims, ylims, ...])

Scatter plot of spatial omics data based on AnnData object.

plot_scatter_img(adata[, xlims, ylims, ...])

Scatter plot of spatial omics data based on AnnData object with optional registered background image.

plot_fov(data, n_fields_x, n_fields_y[, ...])

Plot field of view (FoV) with optional cell highlighting and field partitioning.

plot_network(adata, df[, color_map, ...])

Plot a network graph based on cell type proximity enrichment frequency.

Neighborhoods

calculate_enrichment(adata[, groupby, ...])

Permutation test to calculate proximity enrichment frequency between cell type pairs.

compute_cci(adata, group, sender, receiver)

Infer statistically significant ligand-receptor (LR) pairs by analyzing their co-expression levels within proximal sender-receiver cell pairs.
