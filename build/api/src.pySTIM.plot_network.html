src.pySTIM.plot_network

src.pySTIM.plot_network(adata, df, color_map=None, groupby='celltype', highlight=None, seed=1)[source]

Plot a network graph based on cell type proximity enrichment frequency.

Parameters:

  • adata (AnnData) -- An anndata object containing the data.

  • df (DataFrame) -- DataFrame containing interaction scores between cell types.

  • color_map (dict, optional) -- Dictionary mapping cell types to colors, by default None.

  • groupby (str, optional) -- Column name to group by, by default "celltype".

  • highlight (list, optional) -- List of cell types to highlight, by default None.

  • seed (int, optional) -- Random seed for layout reproducibility, by default 1.

Return type:

None
