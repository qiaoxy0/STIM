src.pySTIM.plot_scatter

src.pySTIM.plot_scatter(adata, xlims=None, ylims=None, color_by=None, genes=None, seed=123, alpha=0.8, cmap=None, ptsize=0.1, ticks=False, dpi=300, width=5, height=5, legend_loc='center left', legend_col=2, bbox_to_anchor=(1.0, 0.5), save=None)[source]

Plot scatter plots based on AnnData object.

Parameters: - adata: A anndata object. - xlims, ylims: The x and y limits for cropping. - color_by: The column to color by. - genes: Gene(s) to plot. - seed: Random seed. - alpha: Transparency level. - cmap: Colormap. - ptsize: Point size. - ticks: Whether to show ticks. - dpi: Dots per inch. - width, height: Width and height of plot. - legend_loc: Legend location. - legend_col: Number of columns in the legend. - bbox_to_anchor: Legend bbox_to_anchor. - save: Filename to save the plot to.

Returns: None

Return type:

None
